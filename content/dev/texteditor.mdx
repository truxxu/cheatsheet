A text editor is fundamental in the webdev workflow. Currently there are many
good alternatives. Some of the most used editors include Atom, Vim, Emacs,
Visual Studio Code and more. In this case, my preference is `Sublime Text`.

```
wget -qO - https://download.sublimetext.com/sublimehq-pub.gpg | sudo apt-key add -
sudo apt install -y apt-transport-https
echo "deb https://download.sublimetext.com/ apt/stable/" | sudo tee /etc/apt/sources.list.d/sublime-text.list
sudo apt update
sudo apt install -y sublime-text
```

Plugins offer new and expanded functionalities. `Package Control` must be intalled
first.
