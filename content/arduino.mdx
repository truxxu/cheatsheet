---
title: "Arduino IDE"
description: "Installing Arduino IDE"
---

1. Download latest Arduino IDE release

2. Extract

```
tar -xvf arduino-1.8.19-linux64.tar.xz
cd arduino-1.8.19
```

3. Run the install script

```
sudo sh install.sh
```

4. Set serial port permissions

```
sudo usermod -a -G dialout _username_
```
